# Memory Template

Use the filename stem as the retrieval key for this memory bank.

For example:
- `james.md` matches the `james` tag.
- `james.research.md` matches both `james` and `research`.
- `client.acme.billing.md` matches `client`, `acme`, and `billing`.

Rules:
- Separate tags with dots.
- Keep the stem to 10 tags or fewer.
- Keep the stem under 100 characters.
- Amy can keep these memories across sessions; the app decides when a turn is memory-worthy and writes it through the file layer.
